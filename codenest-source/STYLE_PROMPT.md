# CodeNest Style Prompt

Create a premium, cinematic landing-page hero for a modern coding education brand. Use a full-bleed dark video background with controlled opacity, left-to-right black-green contrast gradients, and a bottom readability fade. The visual language should feel editorial, technical, restrained, and career-focused rather than playful or startup-generic.

Use a near-black green base (`#070b0a`) with one mint-cyan accent (`#5ed29c`). Add thin architectural grid lines, a wide blurred cyan-green elliptical glow near the upper center, and a single precisely framed liquid-glass information tile. The glass effect should be nearly transparent, with a subtle 4px blur, luminosity blending, a fine white gradient mask border, minimal inset light, and a strict 4px corner radius.

Typography should combine an extra-bold uppercase grotesk headline, compact geometric sans-serif labels, and one controlled italic serif word for editorial contrast. Keep letter spacing neutral, hierarchy dramatic, and supporting copy quiet. Use a rounded mint CTA with dark text and a simple arrow icon. Navigation should be minimal, white, and calm, switching to a full-screen dark mobile menu.

Avoid purple gradients, heavy glassmorphism, decorative orbs, multiple cards, oversized rounded containers, noisy animation, neon cyberpunk effects, and generic three-column startup layouts. Motion should come primarily from the background video and small interaction feedback.

## Long-Page Extension Prompt

Extend the hero into a five-screen editorial landing page without changing the dark visual identity. Borrow the calm pacing of a modern productivity site: let the first content band overlap the hero by roughly 24px, use generous vertical spacing, and alternate between sticky narrative columns and long scrolling content. Keep the page in one dark theme, using near-black green surfaces with subtle tonal changes instead of switching to cream or white sections.

For the project section, create a sticky left column with a large statement, quiet supporting text, and a compact active project navigator. Place three tall translucent project panels on the right, each close to one viewport high, with a real image, a small category marker, a strong editorial title, a restrained metric, and a slow right-to-left reveal. Follow it with a border-led article index, a structured four-stage learning timeline, and a full personal introduction section with a large portrait or workspace image, biography, role, contact information, and a minimal footer.

Use the existing fixed navigation to jump to Projects, Blog, Resume, and About. Preserve one mint accent, neutral letter spacing, strict alignment, thin white borders, soft backdrop blur, restrained motion, and responsive single-column fallbacks. Make section heights configurable and keep all text, media URLs, and personal information in a structured content object that can be edited from a password-gated panel.
